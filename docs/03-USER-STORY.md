# 3. 使用情境 (User Story)

> 維護時可引用：**驗收條件、測試情境、產品／活動說明**。

---

## 情境摘要

在公益活動現場，志工透過平板輸入孩童的夢想與基本資訊，系統經由 AI 生成「未來職人」情境圖與鼓勵小語，並可列印贈送給孩童。

---

## 詳細流程

1. **場景**  
   綠色奇蹟的電腦捐贈活動現場。

2. **互動**  
   志工拿著平板問小朋友：「你長大想做什麼？」

3. **輸入**  
   小朋友說：「我想當醫生，救很多人。」  
   志工在系統輸入「醫生」、「男生」等資訊。

4. **AI 運作**
   * **Gemini**：將描述擴寫為繪圖用 Prompt，例如：「一位穿著白袍的亞洲小男孩醫生，脖子上掛著聽診器，背景是明亮溫暖的醫院，風格為皮克斯動畫風……」
   * **Replicate**：依描述生成對應情境圖像。

5. **結果**  
   螢幕顯示高品質圖片，並附上鼓勵小語，例如：「未來的名醫 [名字]，你的雙手將帶來奇蹟！」

6. **結局**  
   現場可列印贈送給小朋友，作為書桌上的激勵。
